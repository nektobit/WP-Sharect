The plugin allows you to highlight text on your website and share it on social networks

== Description ==

A lightweight (8Kb minified, 2.9Kb gzipped) script to let users share their text selections to social networks, in desktop browsers. (Yes, like Medium).
Based on Sharect.js

* Supports adding custom icons (for now using php snippet)
* You can also control the color of the tooltip widget (also using php code)

<a href="https://nektobit.github.io/WP-Sharect/#documentation">Documentation</a>

# Millstones

The plugin is under active development, I welcome any feedback.

* Add widget color picker
* Adding social media icons from the admin interface
* Optional meta tags

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/sharect-wp` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Go to your site, select any piece of text and make sure the plugin works.

== Frequently Asked Questions ==

= This plugin contains the Facebook and Twitter logos. Do I need special permission to use them on my site? =

You don't need any special permissions. The logos used in this plugin are obtained from official sources that authorize their use and comply with all the style guides of these companies.

= I lack features, can I ask you to add what I need? =

Of course! I am following my development plan, but if your suggestion is useful to all users, I will try to take the time to add it.
